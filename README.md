# 일상게시판
팀원: 오경훈
### 프로젝트 소개
일상의 일들을 자유롭게 적어보는 게시판입니다.
### 개발기간
2023.02.13 ~ 2023.02.21
### 개발환경
-OS: Microsoft Windows 10  
-IDE: Eclipse  
-Database: Oracle 11g Express Edition  
-Web Application Server: Apache Tomcat 9.0  
-Language: Java11, Servlet/JSP, HTML5, CSS3, JavaScript
